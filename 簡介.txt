Dynamic Panel
Inspired by KDE Plasma 6's floating panel design, this extension provides a semi-transparent floating bar effect when there are no windows nearby, and switches to a solid panel style when windows approach. It supports both dark and light modes in GNOME.

Floating Mode
When there are no windows near the top bar, it enters floating mode. In this mode, it only has a semi-transparent effect without blurring (blurring effect is being attempted internally). If you want a blurring effect, it's recommended to use it in conjunction with "Blur my shell" for the panel's static pipeline.

Docked Mode/Solid Mode
When any window gets close enough (almost touching) to the top bar, it transforms into GNOME's default opaque panel, blending seamlessly with maximized windows, unlike themes that always float and might have a "light leak" issue.

Performance Considerations
Due to limitations in GNOME's CSS and GJS, CSS3 transition animations are ineffective for most properties! Also, the requestAnimationFrame function for animation frame synchronization is unavailable. Therefore, smooth movement, resizing, and rounded corner animations have to be achieved through frame-by-frame animations using fixed interval loops, which might impact performance DURING animations. However, there's no performance impact when the panel is static.

Special Thanks
Thanks to the inspiration from "transparent-top-bar". As I'm new to GJS, the implementation of window proximity detection is largely based on the source code of "transparent-top-bar". Without the ideas from this code, this extension would have been difficult to realize.

動態頂部面板
參考KDE Plasma6 的懸浮面板的設計，在附近沒有窗口時呈現半透明懸浮條效果，而當窗口靠近時則呈現實色面板的樣式。支援gnome的暗色模式和亮色模式切換。

懸浮模式
懸浮模式 當topbar附近沒有窗口時，會呈現懸浮模式，懸浮模式僅僅有半透明效果，沒有模糊效果（正在嘗試內建模糊效果），目前如果想要模糊效果，建議搭配Blur my shell對於面板的靜態pipeline使用。

停靠模式/實體模式
停靠模式 當有任何窗口足夠靠近（幾乎接觸）Topbar時，Topbar將會變為gnome默認的不透明的停靠欄，與最大化窗口可以更好的融合，而不像主題一樣始終懸浮會有「漏光」的現象。

關於性能
由於gnome自身的css和gjs的緣故，CSS3的transition補間動畫居然對大部分屬性都是無效的！而requestAnimationFrame這個動畫幀對齊函數也無法使用。因此不得不使用固定幀間隔的循環做逐幀動畫來實現平滑移動、大小改變和圓角動畫，因此在動畫過程中性能上會有一定影響。但在靜止狀態下不存在性能影響。

特別鳴謝
